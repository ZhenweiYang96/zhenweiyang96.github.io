Reseach Projects

Latent Budget Analysis with Neural Network for Compositional Data

The research aims to develop a function in R using the framework of NN and idea of LBA to explore the relationship between several categorical variables.

Multimedia Retrieval

Based on Labeled PSB database, 5 elementary features and 5 distributional features were extracted from the normalized 3D shapes. A user interface was developed to help query similar shapes based on machine learning techniques.

Correction of "treatment paradox" in a time-dependent Cox model

Treatment usage can bias the prediction of "pure" survival probabilities in survival modeling. Thus a simulation study was conducted to test the performance of 6 six existing correction techniques as well as the influencing factors.

Prognostic factors for cognitive function development of Parkinson's disease

Based on PPMI data, a mixed linear model was developed to analyze the cognitive function of patients with Parkinson's disease. The explanatory variables include, complicated diseases and type of anti-hypertension medicine.
